# BFME – Minecraft Mod (NeoForge 1.21.1)

A large-scale Java project inspired by *The Battle for Middle-earth* universe.  
The mod focuses on creating **custom game mechanics**, **faction systems**, and **unique world generation logic** for Minecraft 1.21.1.  
Developed entirely in Java with NeoForge and GeckoLib, this project demonstrates advanced OOP principles and data-driven design.

---

## ⚙️ Main Features
- 🧩 **Custom mechanics** – new crafting systems, workbenches, progression logic
- ⚔️ **Faction system** – internal data model for Gondor, Mordor, and other realms
- 🌍 **Custom world generation** – noise functions, region-based biomes, procedural landscapes
- 🌲 **Custom vegetation and spawners** – tree, density and terrain filters
- 🧱 **Data-driven architecture** – JSON-based configs for easy balancing and expansion
- 🧰 **Debug tools** – runtime logging and generation validation utilities

---

## 🛠 Tech Stack
- **Language:** Java 17
- **Framework:** NeoForge 1.21.1
- **Animation:** GeckoLib
- **Build System:** Gradle
- **IDE:** IntelliJ IDEA
- **Version Control:** Git / GitHub

---

## 🧠 Development Focus
This project explores:
- Advanced **Object-Oriented Programming** and modular code structure
- **Custom registries** and in-game logic systems
- Integration between **data packs**, **resource packs**, and custom code
- Efficient debugging, logging and world-generation profiling
